# SchoolTask

**Tu agenda en orden, tu vida en orden.**

Proyecto web con **Next.js + TypeScript + Tailwind CSS**.

## Requisitos

- Node.js (LTS recomendado)
- npm (o pnpm/yarn/bun)

## Cómo correr

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Qué incluye

- Formulario de alta de producto (nombre, descripción, precio)
- Listado de productos actuales (con total y botón "Quitar")
- Persistencia en el navegador (LocalStorage)
- Branding: paleta de colores, logo (`public/logo.png`), header con eslogan
- Footer con integrantes del proyecto
